complex interactive system

by pianocloud

INITIAL IDEA: A simple interactive game where the player taps to move ROLY out of an unlit maze.

INTERACTIONS

  1. When mouse releases, light illuminates around ROLY for only a few seconds, revealing bits of the maze.
  2. When mouse releases, ROLY moves towards those coordinates and then pauses.
  3. If shape hits an object, then ROLY pauses.
  4. When ROLY touches the blinking light, then the entire canvas is illuminated and revealed.

    5. LEARNINGS: In the end, I was only able to get the visuals in place.
